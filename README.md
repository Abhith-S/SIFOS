# SIFOS
Ethereum Blockchain Voting Dapp


Installations :

1.NodeJS and npm
2.Truffle
3.Ganache
4.Metamask Wallet extension



Running project :

1.Start ganache. Let it run on background.
2.Open command prompt and type :
   git clone https://github.com/AbhithShaji/SIFOS
   cd SIFOS
   npm install
   truffle migrate --reset
   npm run dev
